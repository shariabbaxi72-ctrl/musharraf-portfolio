## ⚙️ Getting Started

### Prerequisites
- Node.js (v18+ recommended)
- npm

### Installation
```bash
git clone https://github.com/shariabbaxi72-ctrl/musharraf-portfolio.git
cd musharraf-portfolio
npm install --legacy-peer-deps
```

### Run locally
```bash
npm run dev
```

### Build for production
```bash
npm run build
```

## 📧 Contact Form Setup (EmailJS)

The contact form uses [EmailJS](https://www.emailjs.com) to send messages directly from the frontend without a custom backend.

Credentials are configured in `ContactForm.jsx`:
- **Service ID**
- **Template ID**
- **Public Key**

Template variables used: `{{name}}`, `{{email}}`, `{{message}}`, `{{time}}`.

> ⚠️ For production, consider moving these IDs into environment variables (`.env`) rather than hardcoding them.
